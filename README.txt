Sistem Pengaduan Sarana Sekolah (PSS) - UKK RPL 2025/2026
=============================================================

1. Deskripsi
   Aplikasi ini adalah sistem berbasis web untuk menangani laporan kerusakan sarana dan prasarana di lingkungan sekolah.

2. Teknologi
   - PHP Native
   - MySQL
   - CSS (Custom Modern UI)

3. Instalasi
   - Copy folder 'PSS jier' ke htdocs (C:\xampp\htdocs\).
   - Buat database baru di PHPMyAdmin dengan nama 'pss_jier'.
   - Import file 'database.sql' yang ada di dalam folder proyek.
   - Akses di browser: http://localhost/PSS jier

5. Fitur Utama
   - Login Siswa & Admin (Multi-role)
   - Dashboard Statistik
   - Tulis Aspirasi dengan Foto
   - Riwayat Aspirasi
   - Tanggapan Admin
   - Filter & Cetak Laporan
   - Keamanan (MD5, Validasi Input, Session)
